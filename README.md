- esx/qbcore
- oxmysql

# Features
- Custom UI
- Configurable
- Optimized (0.0ms idle)

# Installation
1. Drag `Ricky-Reward` in your resource folder
2. Execute SQL File
3. Setup config
4. Add `ensure Ricky-Reward` in your server.cfg
5. Start the server.
